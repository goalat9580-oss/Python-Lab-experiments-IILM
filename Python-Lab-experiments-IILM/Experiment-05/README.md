## Problem Statement
Write a program to find and return the **largest element** in a given array.

**Input:**  
Array: [1, 8, 7, 56, 90]

**Output:**  
The largest element in the array is 90

---

## Algorithm
1. Take the input array.  
2. Initialize a variable `max_element` with the first element of the array.  
3. Traverse the array and compare each element with `max_element`.  
4. If the current element is greater than `max_element`, update `max_element`.  
5. After traversal, print `max_element`.
